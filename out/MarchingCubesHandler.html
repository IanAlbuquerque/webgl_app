MarchingCubesHandler

MarchingCubesHandler

new MarchingCubesHandler()

The handler of the Marching Cubes' algorithm. Contains the methods necessary for running the algorithm.

Members

private edges_triangle_table :Array.<number>

Marching Cubes Reference: http://fab.cba.mit.edu/classes/S62.12/docs/Lorensen_marching_cubes.pdf Table obtained in the following link: http://graphics.stanford.edu/~mdfisher/MarchingCubes.html Table containing the edges that will compose each triangle of all the 256 marching cube cases. Every 3 values represents a triangle. Edges are numbered from 0 to 11.
