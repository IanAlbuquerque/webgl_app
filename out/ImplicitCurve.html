ImplicitCurve

ImplicitCurve

new ImplicitCurve(xMin, xMax, yMin, yMax, numPontos, equation)

A implicit curve in R^2

Parameters:
Name Type Description
xMin number The left bound of the domain.
xMax number The right bound of the domain.
yMin number The bottom bound of the domain.
yMax number The top bound of the domain.
numPontos number The number of points to be used for the function drawing discretization.
equation string A string containing the script that is evaluated as the equation of the implicit curve. The x and y variables must be used.

Members

equation

The array containing the coefficients of the polinomio. The entry coefficients[i] corresponds to the coefficient of x^i.
Default Value:
  • "x*x+y*y-1"

numPontos

The number of points to be used for the function drawing discretization.
Default Value:
  • 40

xMax

The right bound of the domain.
Default Value:
  • 1

The left bound of the domain.

calculateDrawingPoints() → {Array.<number>}

Draws the implicit curve in the plane XY using the algorithm "Marching Triangles"

Returns:
Array.<number> The list of points that represents the drawing. Every two entries represents the x and y informations of a point. The drawing should be made using gl.LINES.
