webglut/gl_module

webglut/gl_module

The module responsible for initializing the gl global object.

Methods

inner initialize(canvas)

Initializes the gl associating it to a canvas element.

Parameters:
Name Type Description
canvas Canvas The canvas element to associate the gl.
Throws:
Type
Exception
Example
// Gets the document element with the id "main-canvas"
var canvas = document.getElementById("main-canvas");
// Initialize the gl global object using a "webgl" context associated to the canvas selected
GLModule.initialize(canvas);
