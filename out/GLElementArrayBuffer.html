GLElementArrayBuffer

GLElementArrayBuffer

new GLElementArrayBuffer(data, itemSize, numItems)

A webgl element array buffer. Used to store a variety of data in a index-like format needed for redering scenes.

Parameters:
Name Type Description
data Array.<number> The array containing the data to be inserted into the buffer.
itemSize number The number of entries in the array that defines an unique element.
numItems number The number of unique elements in the array.
