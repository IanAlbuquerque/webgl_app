FunctionR2

FunctionR2

new FunctionR2(xMin, xMax, numPontos, fOfX)

A function defined from R to R.

Parameters:
Name Type Description
xMin number The left bound of the domain.
xMax number The right bound of the domain.
numPontos number The number of points to be used for the function drawing discretization.
fOfX string A string containing the script that is evaluated as f(x). The x variable must be used.
Example
// Function sin(x).
var myFunction = new FunctionR2(-1,1,500,"Math.sin(x)");
// Draws the function.
myFunction.draw();

Members

fOfX

The array containing the coefficients of the polinomio. The entry coefficients[i] corresponds to the coefficient of x^i.

numPontos

The number of points to be used for the function drawing discretization.

xMax

The right bound of the domain.
Default Value:
  • 1

xMin

The left bound of the domain.
Default Value:
  • -1

Draws the function in the plane XY.

Array.<number> The list of points that represents the drawing. Every two entries represents the x and y informations of a point. The drawing should be made using gl.LINE_STRIP.
