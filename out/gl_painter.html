Methods

inner begin(primitive)

Begin drawing a primitive utilizing a vertex buffer. It's expected to be followed by vertex2d or vertex3d as well as end.

Parameters:
Name Type Description
primitive GLPrimitive The primitive being utilized for drawing the following vertices.
Example
// Sets the current drawing color to red.
GLPainter.setDrawColor([1,0,0,1]);

// Begin drawing a line
GLPainter.begin(gl.LINES);
	// Adds first vertex (-1,0) to the buffer
	GLPainter.vertex2d(-1,0);
	// Adds second vertex (1,0) to the buffer
	GLPainter.vertex2d(1,0);
// Finishes drawing the line (actually, only after this line of code the line is drawn)
GLPainter.end();

inner end()

Draws the vertices stored in the internal vertex buffer. It's expected to be preceeded by begin, and the commands vertex2d or vertex3d.

Example
// Sets the current drawing color to red.
GLPainter.setDrawColor([1,0,0,1]);

// Begin drawing a line
GLPainter.begin(gl.LINES);
	// Adds first vertex (-1,0) to the buffer
	GLPainter.vertex2d(-1,0);
	// Adds second vertex (1,0) to the buffer
	GLPainter.vertex2d(1,0);
// Finishes drawing the line (actually draws the line)
GLPainter.end();

inner setDrawColor(color)

Sets the current color being used for the drawings.

Parameters:
Name Type Description
color Array.<number> The new color to be utilized for the drawing. Uses the RBGA system. Expects 4 numbers between or equal to 0 and 1 representing the colors Red, Green, Blue and the opacity Alpha, in this order.
Examples
// Sets the current drawing color to red.
GLPainter.setDrawColor([1,0,0,1]);
// Sets the current drawing color to blue.
GLPainter.setDrawColor([0,0,1,1]);
// Sets the current drawing color to green.
GLPainter.setDrawColor([0,1,0,1]);
// Sets the current drawing color to pink.
GLPainter.setDrawColor([1,0,1,1]);
// Sets the current drawing color to grey.
GLPainter.setDrawColor([0.5,0.5,0.5,1]);
// Sets the current drawing color to red.
GLPainter.setDrawColor([1,0,0,1]);

// Begin drawing a line
GLPainter.begin(gl.LINES);
	// Adds first vertex (-1,0) to the buffer
	GLPainter.vertex2d(-1,0);
	// Adds second vertex (1,0) to the buffer
	GLPainter.vertex2d(1,0);
// Finishes drawing the line (actually, only after this line of code the line is drawn)
GLPainter.end();
