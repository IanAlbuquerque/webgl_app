ImplicitCurve

ImplicitCurve

new ImplicitCurve(_x_min, _x_max, _y_min, _y_max, _implicit_curve_expression)

An implicit curve in the XY plane.

Parameters:
Name Type Description
_x_min number The left bound of the domain.
_x_max number The right bound of the domain.
_y_min number The bottom bound of the domain.
_y_max number The top bound of the domain.
_implicit_curve_expression string A string containing the script that is evaluated as the expression of the implicit curve. The associated equation of the implicit curve will be considered as "implicit_curve_expression=0". The x and y variables must be used.

generateDrawingVertices(number_of_points_x, number_of_points_y) → {Array.<number>}

Generates the vertices of the implicit curve discretization in the plane XY using the algorithm "Marching Triangles" The vertices are supposed to be drawn using the LINES primitive.

Parameters:
Name Type Description
number_of_points_x number The number of points to be used for the first coordinate discretization.
number_of_points_y number The number of points to be used for the second coordinate discretization.
Returns:
Array.<number> generated_vertices The list of points that represents the drawing. Every two entries represents the x and y coordinates of a vertex.
