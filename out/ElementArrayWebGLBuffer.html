ElementArrayWebGLBuffer

ElementArrayWebGLBuffer

new ElementArrayWebGLBuffer(webgl_context, _data, _item_size, _number_of_items)

A WebGL element array buffer. Used to store a variety of index-like data needed for setting up scenes.

Parameters:
Name Type Description
webgl_context WebGLContext The WebGL canvas context that will be used for loading the matrices.
_data Array.<number> The array containing the data to be inserted into the buffer.
_item_size number The number of entries in the array that defines an unique element.
_number_of_items number The number of unique elements in the array.
