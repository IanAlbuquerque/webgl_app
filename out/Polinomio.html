Polinomio

Polinomio

new Polinomio(xMin, xMax, numPontos, coefficients)

A polinomio.

Parameters:
Name Type Description
xMin number The left bound of the domain.
xMax number The right bound of the domain.
numPontos number The number of points to be used for the function drawing discretization.
coefficients Array.<number> An array containing the coefficients of the polinomio. The entry coefficients[i] corresponds to the coefficient of x^i.
Example
// Polinomio x^2.
var myPolinomio = new Polinomio(-1,1,500,[0,0,1]);
// Draws the polinomio.
myPolinomio.draw();
// Draws 2*x, the derivative of the polinomio
myPolinomio.derivative().draw();
